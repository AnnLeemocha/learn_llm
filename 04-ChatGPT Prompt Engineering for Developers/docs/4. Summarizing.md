# Summarizing 總結
使用大型語言模型進行文字摘要

為什麼要摘要？
* 世界上文字太多、時間太少，人無法讀完所有資訊。
* 文字摘要是一個超級實用的應用，可以快速抓住重點。
* 常見應用場景：新聞摘要、文章摘要、產品評論摘要、會議記錄摘要等。

如果你在開發的應用中也有大量文字內容，你可以怎麼設計提示來摘要這些資訊，幫助使用者快速了解文字的重點，必要時再深入查看原始內容。

## 待總結文本
如果您正在建立電子商務網站，並且有大量的評論，那麼擁有一個可以總結長篇評論的工具可以讓您快速瀏覽更多評論，從而更好地了解所有客戶的想法。

以總結該產品評論的任務作為運行範例。
```python
prod_review = """
Got this panda plush toy for my daughter's birthday, \
who loves it and takes it everywhere. It's soft and \ 
super cute, and its face has a friendly look. It's \ 
a bit small for what I paid though. I think there \ 
might be other options that are bigger for the \ 
same price. It arrived a day earlier than expected, \ 
so I got to play with it myself before I gave it \ 
to her.
"""
```

## 以單字/句子/字元限制進行總結
可以控製字元數或句子數等來影響摘要的長度。
```python

# 您的任務是產生來自電子商務網站的產品評論的簡短摘要。
#
# 總結下面的評論，用三個反引號分隔，最多 30 個字。
#
# 評論：...
# ---------------------------------------------------------
prompt = f"""
Your task is to generate a short summary of a product \
review from an ecommerce site. 

Summarize the review below, delimited by triple 
backticks, in at most 30 words. 

Review: ```{prod_review}```
"""

response = get_completion(prompt)
print(response)

# ---------------------------------------------------------
# 絨毛柔軟可愛，女兒喜歡，價格略高，但提前送達。 
# ---------------------------------------------------------
```

## 有特定目的要產出摘要
可以針對不同部門，設計不同的摘要用途，提高溝通效率。

### 重點總結"運輸和交付"
* 物流部門
```python
# ---------------------------------------------------------
# 您的任務是產生來自電子商務網站的產品評論的簡短摘要，以便向運輸部門提供回饋。
# 
# 總結下面的評論，用三個反引號分隔，最多 30 個字，並重點關注提及產品運輸和交付的任何方面。
# 
# 評論：...
# ---------------------------------------------------------
prompt = f"""
Your task is to generate a short summary of a product \
review from an ecommerce site to give feedback to the \
Shipping deparmtment. 

Summarize the review below, delimited by triple 
backticks, in at most 30 words, and focusing on any aspects \
that mention shipping and delivery of the product. 

Review: ```{prod_review}```
"""

response = get_completion(prompt)
print(response)

# ---------------------------------------------------------
# 產品比預期早一天送達。
# ---------------------------------------------------------
```

### 重點總結"價格和價值"
* 定價部門
```python
# ---------------------------------------------------------
# 您的任務是產生來自電子商務網站的產品評論的簡短摘要，以便向負責確定產品價格的定價部門提供回饋。
# 
# 總結下面的評論，用三個反引號分隔，最多 30 個字，並重點關注與價格和感知價值相關的任何方面。
#
# 評論：...
# ---------------------------------------------------------
prompt = f"""
Your task is to generate a short summary of a product \
review from an ecommerce site to give feedback to the \
pricing deparmtment, responsible for determining the \
price of the product.  

Summarize the review below, delimited by triple 
backticks, in at most 30 words, and focusing on any aspects \
that are relevant to the price and perceived value. 

Review: ```{prod_review}```
"""

response = get_completion(prompt)
print(response)

# ---------------------------------------------------------
# 對這個尺寸來說價格略高。
# ---------------------------------------------------------
```

## 嘗試"擷取"而不是"摘要"
根據需求選擇濃縮還是萃取。
* 摘要（Summarize）：
  濃縮內容但保留敘述語氣，可能包含多項資訊。
  (包括與焦點主題無關的主題)
* 擷取（Extract）：
  只擷取最關鍵的片段或事實。
  
完全排除不相關資訊，而不只是摘要，你也可以把提示改成「提取資訊」而非摘要。
```python
# ---------------------------------------------------------
# 您的任務是從電子商務網站的產品評論中提取相關信息，以便向運輸部門提供回饋。
# 
# 從下面的評論中，用三重引號分隔提取與運輸和交付相關的資訊。限制為 30 個字。
# 
# 評論：...
# ---------------------------------------------------------
prompt = f"""
Your task is to extract relevant information from \ 
a product review from an ecommerce site to give \
feedback to the Shipping department. 

From the review below, delimited by triple quotes \
extract the information relevant to shipping and \ 
delivery. Limit to 30 words. 

Review: ```{prod_review}```
"""

response = get_completion(prompt)
print(response)

# ---------------------------------------------------------
# 產品比預期早一天送達。
# ---------------------------------------------------------
```

## 總結多個產品評論
適用於多篇評論、自動生成摘要列表，以便更快速瀏覽。
* 能快速瀏覽大量評論的重點。
* 可以構建儀表板（dashboard），快速瀏覽所有評論的精華摘要，節省客服與產品人員時間。

這些評論都不短，如果你想知道評論者的主要反饋，又不想逐一細讀，那該怎麼辦？

我們可以把每一則評論放進一個列表，然後用迴圈處理它們。
